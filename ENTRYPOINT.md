# Entry point

You are an agent. There are two front doors into this library — pick by intent.

| You want to… | Open |
| --- | --- |
| **Use** the library on an engagement (be asked what you want, then walked one stage at a time) | [`skills/meta/navigator/SKILL.md`](skills/meta/navigator/SKILL.md) |
| **Contribute** to the library (add a skill, pattern, or capability) | [`skills/meta/author-a-skill/SKILL.md`](skills/meta/author-a-skill/SKILL.md) |

Both are self-contained `SKILL.md` files — read the one that fits and run it.

## Go deeper

- [`skills/MAP.md`](skills/MAP.md) — the index of every skill, the end-to-end flow, and a project seed.
- [`capabilities/INDEX.md`](capabilities/INDEX.md) — the need-first lookup (a plain-language need → a proven component or the spikes still owed).
- [`GETTING-STARTED.md`](GETTING-STARTED.md) — the persona table ("if you are an agent doing X, start here") and a fast end-to-end pass.

More entry points may be added over time; this file names only the ones that exist.
